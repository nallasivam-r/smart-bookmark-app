## Problems I Faced & Solutions

### 1. Git commit error
Error: switch `m` requires a value  
Solution: Added commit message using `git commit -m "Initial commit"`

### 2. Favicon not updating
Solution: Replaced favicon.png in public folder and restarted dev server.

### 3. Deployment issues
Fixed environment variables in Vercel dashboard.
